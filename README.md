# Harvard Media — Sales Dashboard V1

Internal sales performance dashboard for Harvard Media. Covers revenue (fiscal YTD), CRM account hygiene, ask and activity metrics, and AI-generated coaching insights.

---

## Project structure

```
Sales-Dashboard-V1/
├── index.html                  ← Main dashboard page
├── README.md                   ← This file
├── data/
│   └── dashboard_data.json     ← Data file (updated daily)
└── assets/
    ├── style.css               ← Dashboard styles
    └── dashboard.js            ← Dashboard logic
```

---

## Data sources

| File | Source | Frequency |
|------|--------|-----------|
| `dashboard_data.json` | Generated by `update_dashboard.py` | Daily |
| MyCustomers export | Rumple CRM | Daily |
| Activity report | Rumple CRM | Monthly |
| Report 4 (YTD revenue) | SDS | Daily |

---

## How to update the dashboard daily

1. Export fresh files from Rumple and SDS
2. Drop all three files into the same folder as `update_dashboard.py`
3. Double-click `update_dashboard.py`
4. Upload the new `data/dashboard_data.json` to this repo
5. Dashboard updates automatically

---

## Data definitions

- **Active accounts:** Pitched within the last 6 weeks (Rumple definition)
- **Inactive accounts:** Not pitched in the last 6 weeks
- **Never pitched:** No Last Pitched date in Rumple
- **Zero activity:** No note, call, appointment, CNA, or pitch ever logged
- **Close rate:** Asks with closed amount > $0 ÷ total asks created
- **Revenue period:** Sep 2025 – current month (fiscal year)
